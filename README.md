# 💴 MoneyFlow

個人事業主向けキャッシュフロー可視化アプリ。

## 機能

- **月カレンダービュー** — 日別の収支・「今日時点で使える金額」を表示
- **危険ゾーン警告** — 残高がマイナスになる日を自動検出
- **収支管理** — 確定/見込み収入、固定費、スポット支出（消費税・車検など）
- **税金バッファ** — 収入の○%を税金用に自動控除
- **欲しいものリスト** — 今月の自由資金で買えるか判定

## セットアップ

```bash
# 1. このフォルダに移動
cd moneyflow

# 2. 依存関係をインストール
npm install

# 3. 開発サーバー起動
npm run dev
```

ブラウザで `http://localhost:5173` を開く。

## データについて

データはブラウザの **localStorage** に自動保存されます。
ブラウザを閉じても消えません。

## ビルド（本番用）

```bash
npm run build
```

`dist/` フォルダに静的ファイルが出力されます。
